Assignment 1 handin

How to run:
- Open `Assignment 1/Handin/feedforwardAssignment.ipynb` in Jupyter Notebook or JupyterLab and run the cells.

Requirements:
- Python 3.8+
- numpy, torch, matplotlib, pandas, notebook

Quick setup (PowerShell):
```
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r "Assignment 1/Handin/requirements.txt"
jupyter notebook
```

Notes:
- Frames from Task 3.4 are saved to `Assignment 1/Handin/frames_task3_4/` by the notebook code.
- For GPU support install the correct PyTorch wheel from https://pytorch.org.
